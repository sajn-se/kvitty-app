# Swedish Terminology Reference

This is a Swedish accounting application. Key terminology used throughout the codebase:

## Business Types
- **Aktiebolag (AB)** = Limited company
- **Enskild firma** = Sole proprietorship
- **Handelsbolag (HB)** = General partnership
- **Kommanditbolag (KB)** = Limited partnership
- **Ekonomisk förening** = Economic association
- **Ideell förening** = Non-profit association
- **Stiftelse** = Foundation

## Accounting Terms
- **Faktura** = Invoice
- **Kund** = Customer
- **Produkt** = Product
- **Moms** = VAT (Value Added Tax)
  - 25% = Standard rate
  - 12% = Reduced rate (mat, hotell)
  - 6% = Reduced rate (tidningar, böcker, transport)
  - 0% = Zero-rated (export)
- **Verifikation** = Accounting verification/voucher/journal entry
- **Räkenskapsår** = Fiscal year
- **Bokslut** = Annual closing
- **Resultaträkning** = Income statement
- **Balansräkning** = Balance sheet
- **BAS-kontoplan** = Swedish standard chart of accounts

## Payroll Terms
- **Lön** = Salary/Payroll
- **Bruttolön** = Gross salary
- **Nettolön** = Net salary
- **Arbetsgivaravgifter** = Employer contributions (31.42% standard)
- **Skatteavdrag** = Tax deduction
- **AGI** = Arbetsgivardeklaration (employer tax declaration to Skatteverket)
- **Preliminärskatt** = Preliminary tax
- **Lönespecifikation** = Salary statement

## Tax & Deductions
- **ROT** = Renovation, Ombyggnad, Tillbyggnad (30% deduction)
- **RUT** = Hushållsnära tjänster (50% deduction)
- **Skatteverket** = Swedish Tax Agency
- **Vinstmarginalbeskattning** = Margin scheme (used goods taxation)
- **Omvänd skattskyldighet** = Reverse charge (EU B2B)

## Payment Terms
- **Bankgiro** = Swedish bank clearing system
- **Plusgiro** = Swedish postal clearing system
- **Swish** = Swedish mobile payment system
- **OCR** = Optical Character Recognition number (payment reference)
- **Förfallodatum** = Due date
- **Betalningsvillkor** = Payment terms

## Document Types
- **Kvitto** = Receipt
- **Inkomst** = Income
- **Leverantörsfaktura** = Supplier invoice
- **Utlägg** = Out-of-pocket expense

## Product Units
- **Styck** = Pieces/Items
- **Timmar** = Hours
- **Dagar** = Days
- **Månader** = Months
- **Kilogram** = Kilograms
- **Meter** = Meters
- **M²** = Square meters
- **M³** = Cubic meters

## Report Periods
- **Månatlig** = Monthly
- **Kvartalsvis** = Quarterly
- **Årlig** = Annual
